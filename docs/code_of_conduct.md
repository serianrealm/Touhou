# Contributor Covenant 3.0 Code of Conduct

## Our Pledge

We pledge to make participation in this project welcoming, safe, and respectful for everyone.

We are committed to fostering an environment that values dignity, constructive collaboration, and technical discussion grounded in respect for others.

## Encouraged Behavior

- be respectful and direct
- assume good intent, but communicate clearly when something is broken or risky
- give and receive constructive feedback
- take responsibility for your own words, actions, and follow-up
- keep disagreements technical and avoid personal attacks

## Unacceptable Behavior

- harassment, intimidation, or threats
- discriminatory language or behavior
- personal insults, character attacks, or repeated hostile conduct
- publishing private information without permission
- deliberate disruption of project spaces or review processes

## Scope

This Code of Conduct applies in project spaces, including:

- issues
- pull requests
- code review
- discussions
- chat or email when representing the project

## Reporting

To report a possible violation, email:

- `serianrealm@gmail.com`

Please include any relevant context, logs, screenshots, or links.

## Enforcement

Maintainers may take any action they consider appropriate, including:

- private warning
- temporary communication limits
- temporary suspension from project spaces
- permanent removal from project spaces

Enforcement decisions will be made with the goal of protecting contributors and keeping the project workable.
